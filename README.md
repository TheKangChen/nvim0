# Neovim Zero

A zero/minimal plugin neovim config.

Created as a **minimal plugin** version of my existing Neovim config.

## Plans

This config is great as is but for the ease of use, I plan on adding [mini.nvim](https://github.com/nvim-mini/mini.nvim) plugins in the future.
